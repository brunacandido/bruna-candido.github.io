# Bruna Cândido — Portfolio

Personal portfolio of **Bruna Cândido**, Geodata Scientist and Oceanographer specializing in AI for Earth Observation and remote sensing.

**Live:** [brunacandido.github.io/bruna-candido.github.io](https://brunacandido.github.io/bruna-candido.github.io/)

---

## About

The portfolio showcases:
- Research and academic projects from the [Copernicus Master in Digital Earth](https://www.master-cde.eu) (Erasmus Mundus)
- Featured project: SAR-based oil spill detection with deep learning (internship at CMCC)
- Skills in Python, geospatial tools, and Earth Observation platforms (Sentinel-1/2, GEE, CMEMS)
- Education and work experience

---

## Tech Stack

- **React 18** + **TypeScript**
- **SCSS** — custom design system with OKLCH color tokens and fluid typography
- **Deployed** via GitHub Pages (`gh-pages`)

---

## Local Development

```bash
npm install --legacy-peer-deps
npm start
```

Opens at [http://localhost:3000](http://localhost:3000).

## Build & Deploy

```bash
npm run build    # production build
npm run deploy   # push to GitHub Pages (master branch)
```
